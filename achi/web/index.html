ℹ️ 关于 Achievement

Achievement 是一个基于 AI 的求职辅助工具，帮助你：

  • ✏️ 简历优化 - 根据目标岗位 JD 优化简历，提高面试邀约率
  • 🎯 岗位匹配 - AI 猎头顾问智能匹配最适合你的岗位
  • 🕷️ 职位爬虫 - 实时爬取 51job 职位数据
  • 📄 简历浓缩 - 使用 RAG 技术快速生成简历摘要
  • 📋 投递跟踪 - 记录和管理求职投递历史
  • 📎 文件解析 - 解析 PDF/Word 简历文件

技术栈：Python MCP · DeepSeek API · Qdrant 向量数据库 · SQLite
