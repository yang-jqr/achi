## 综合求职建议

### 1. 突出 AI Agent 与 RAG 项目实战
在简历和面试中，重点描述您主导或参与的 AI Agent 项目（如智能问答、自动规划、工具调用等），以及 RAG 系统的构建（如文档检索、知识库增强、Prompt 优化）。**可量化成果**（例如：检索准确率提升 X%，任务完成率提高 Y%）会大幅提升竞争力。

### 2. 补充系统设计与架构能力
腾讯、拼多多的岗位都要求"架构设计经验"，建议您准备 1-2 个高并发、分布式场景下的 AI Agent 架构案例（如多 Agent 协作、Agent 与外部工具/API 的集成、缓存与限流策略等）。如果暂时缺少这类经验，可学习 **LangGraph、CrewAI** 等框架的架构设计。

### 3. 技能栈微调：熟悉 Java/Golang 加分
虽然 AI Agent 开发常用 Python，但阿里云岗位要求 Java/Golang，拼多多和腾讯也接受多种语言。若您时间允许，快速上手 Java 或 Golang 的基础语法，并在简历中标注"熟悉 Java/Golang 开发（5年经验中的部分项目使用）"，将扩大岗位覆盖面。

### 4. 薪资谈判策略
您期望 30K，上述岗位均提供 30K 以上甚至更高，但部分岗位（如阿里云）薪资范围为 30-50K，建议您可以用"当前期望 30K，但更看重平台和成长"为切入点，争取中上水平（35-40K）。拼多多和腾讯的 16 薪模式实际月薪可拆解为更高年薪。

### 5. 投递顺序
1. **优先投递**：拼多多 AI Agent 开发（直接对口，且安全方向是当前热点）
2. **次优投递**：腾讯 AI Agent 后台开发（品牌影响力大）
3. **第三选择**：阿里云 TAM（需要了解业务支持与客户对接，但技术深度依然匹配）
4. **保底选项**：珍岛集团 AI Agent 开发（20-30K）
